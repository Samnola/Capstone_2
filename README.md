## Southern Sales Territory Analysis 

This project analyzes sales data for EmporiUm, a bookstore chain that sells books, tech gear and art supplies. 

This analysis compares sales performance between the **Florida** and **Texas** sales territories.
The goal of this project is to identify sales trends, compare territory performance, and better understand store revenue across both regions.


## Key Findings

- Technology & Accessories consistently generated the highest revenue across both territories.
- Q3 analysis (July–September) over the last two years showed stable and repeatable demand.
- A small number of stores significantly outperformed others in total sales.
- Florida and Texas showed similar performance patterns during Q3
